CHICHI'S COLOR GAME — iOS Xcode Project
========================================
Open ChichiGame.xcodeproj in Xcode on a Mac (requires Xcode 15+, iOS 15+).

1. Open ChichiGame.xcodeproj
2. Select your team under Signing & Capabilities
3. Choose your device or simulator
4. Press Run (Cmd+R)

The app wraps the game in a full-screen WKWebView.
All web assets are bundled in ChichiGame/web/.

Requirements:
  - macOS with Xcode 15 or later
  - iOS 15.0+ deployment target
  - Apple Developer account (for device testing / App Store)
